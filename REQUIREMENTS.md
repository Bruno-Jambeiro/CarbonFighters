# Elicitação de Requisitos

Para esta fase do projeto Carbon Fighters, foram utilizadas duas técnicas de elicitação de requisitos:

- Entrevistas (Estrutura Funil): para compreender diretamente as necessidades e expectativas dos usuários potenciais.
- Benchmarking: para analisar como plataformas semelhantes abordam a conscientização ambiental e a gamificação de ações sustentáveis.

O objetivo geral desta etapa foi identificar requisitos funcionais e não funcionais que servirão de base para a definição dos épicos e histórias de usuário na fase seguinte do projeto.

## Entrevista - Estrutura Funil

### 1. Introdução

As entrevistas foram realizadas de forma online com potenciais usuários do sistema, buscando compreender o comportamento, as motivações e as dificuldades enfrentadas em relação à adoção de práticas sustentáveis no dia a dia.
Uma estrutura tipo funil foi utilizada para estruturar as perguntas de maneira progressiva — partindo do contexto geral até chegar a operações e requisitos específicos.

### 2. Objetivo da entrevista

O objetivo desta entrevista é compreender o contexto, os objetivos, as necessidades e as operações relacionadas ao projeto Carbon Fighters, identificando informações que permitam a elicitação de requisitos para o desenvolvimento de uma solução tecnológica voltada à conscientização ambiental e redução de emissão de carbono.
